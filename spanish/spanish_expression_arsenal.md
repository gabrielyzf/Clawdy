# Spanish Expression Arsenal — Frases Fijas

_High-utility fixed phrases, collocations, sentence frames, discourse markers._

Format:
```
## "frase fija"
- Meaning (EN): ...
- Meaning (CN): ... (optional)
- Use case: ...
- Example 1: ...
- Example 2: ...
- Gabriel's example: ...
- Common mistake: ...
- Added: YYYY-MM-DD
- Last reviewed: YYYY-MM-DD
- Next review: YYYY-MM-DD
- Status: new / learning / known / mastered
```

Review intervals: 1d → 3d → 7d → 14d → 30d.
