# Personal Intelligence System

## Overview

A personal AI-powered knowledge and intelligence system for Gabriel. Captures, processes, and surfaces information from various sources (WeChat articles, web, notes, etc.) into a structured, queryable knowledge base.

## Goals

- Ingest content from multiple sources (WeChat, web, manual notes)
- Structure and store in Notion and/or vector DB (RAG)
- Enable AI-assisted retrieval, synthesis, and insight generation
- Automate recurring research and briefing workflows
